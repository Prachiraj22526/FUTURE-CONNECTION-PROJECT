This project aims to predict the future type of electricity connection that a customer may opt for, using Machine Learning algorithms. The system is developed based on real-world data from North Bihar Power Distribution Company Limited (NBPDCL).

The model analyzes various features such as:
- Customer consumption patterns
- Load usage history
- Location and category of users
- Past connection types

Using algorithms like Random Forest and the system classifies whether a customer will require a domestic, commercial, or industrial connection in the future.

Objectives:
- Assist NBPDCL in demand forecasting
- Improve resource allocation and infrastructure planning
- Enable efficient load management
- Support smart grid development

Technologies Used:
- Python
- Pandas, NumPy
- Scikit-learn
- Machine Learning Models (Random Forest)# FUTURE-CONNECTION-PROJECT
